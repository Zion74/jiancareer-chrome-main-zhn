## 🚀 Chrome Web Store 发布步骤

### 1. 开发者账户准备
1. 访问 [Chrome Web Store Developer Dashboard](https://chrome.google.com/webstore/devconsole/)
2. 使用Google账户登录
3. 支付一次性注册费用（$5 USD）
4. 完成开发者身份验证

### 2. 上传扩展包
1. 点击「新增项目」
2. 上传 `JianCareer-AutoFill-Extension.zip` 文件
3. 等待自动验证完成

### 3. 填写商店信息

#### 基本信息
- **名称**：JianCareer AutoFill - 简职自动填写
- **摘要**：一键填写求职信息，让求职更简单高效
- **详细描述**：
```
简职自动填写扩展是一款专为求职者设计的Chrome浏览器扩展，支持在主流招聘网站上一键填写个人信息和简历内容。

🎯 主要功能：
• 支持Boss直聘、拉勾网、前程无忧、猎聘网等主流招聘平台
• 智能识别表单字段，精准填写个人信息
• 多版本简历管理，适应不同职位需求
• 安全加密存储，保护个人隐私
• 简洁易用的界面设计

✨ 使用场景：
• 批量投递简历时快速填写重复信息
• 在线申请职位时自动填充个人资料
• 提高求职效率，节省宝贵时间

🔒 隐私保护：
• 所有数据本地存储，不上传到服务器
• 支持数据加密，确保信息安全
• 用户完全控制数据的使用和删除
```

### 5. 隐私和权限说明

#### 权限说明模板
```
本扩展需要以下权限来提供服务：

• storage：本地存储用户的简历信息和设置
• activeTab：获取当前标签页信息以识别招聘网站
• scripting：在招聘网站页面中注入自动填写脚本
• host权限：访问主流招聘网站以提供自动填写功能

所有权限仅用于核心功能，不会收集或传输用户隐私数据。
```

## 🔧 常见问题解决

### 审核被拒常见原因
1. **权限过度**：确保只申请必需的权限
2. **描述不清**：详细说明扩展功能和权限用途
3. **截图质量**：提供清晰、高质量的功能截图
4. **隐私政策**：如收集用户数据需提供隐私政策链接

### 技术问题排查
1. 检查manifest.json语法
2. 验证所有文件路径正确
3. 测试在不同网站的兼容性
4. 确保CSP策略不会阻止功能
